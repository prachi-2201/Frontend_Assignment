                        // Function to calculate days left until next Christmas
                        function daysUntilChristmas() {
                            // Get the current date
                            const currentDate = new Date();

                            // Get the current year
                            const currentYear = currentDate.getFullYear();

                            // Set the date for Christmas this year
                            const christmasDate = new Date(currentYear, 11, 25); // Months are 0-based, so 11 represents December

                            // Check if Christmas has already passed this year, if so, set the date for Christmas next year
                            if (currentDate > christmasDate) {
                                christmasDate.setFullYear(currentYear + 1);
                            }

                            // Calculate the difference in days
                            const daysLeft = Math.ceil((christmasDate - currentDate) / (1000 * 60 * 60 * 24));

                            return daysLeft;
                        }

                        // Example usage
                        const daysLeftUntilChristmas = daysUntilChristmas();
                        document.write(`There are ${daysLeftUntilChristmas} days left until next Christmas.`);
Q.10 What is Condition Statement?

The if/else statement is a part of JavaScript's "Conditional" Statements, which are used to perform different actions based on different conditions.

In JavaScript we have the following conditional statements:

  • Use if to specify a block of code to be executed, if a specified condition is true

  • Use else to specify a block of code to be executed, if the same condition is false

  • Use else if to specify a new condition to test, if the first condition is false

  • Use switch to select one of many blocks of code to be executed

  • 5 > 3 evaluates to true.

true && true evaluates to true.

Q.17 What is the result of the expression true && false || false && true?

The expression true && false || false && true involves the logical AND (&&) and logical OR (||) operators.

In JavaScript, the logical AND operator (&&) has higher precedence than the logical OR operator (||). Therefore, the expression is evaluated from left to right, and the result is determined accordingly.

Let's break down the expression:

  1. true && false evaluates to false (because both conditions must be true for the AND operator to result in true).
  2. false || false evaluates to false (because at least one condition must be true for the OR operator to result in true).
  3. Finally, false && true also evaluates to false.
  4. So, the result of the entire expression is false.

Q.20 What is the difference between && and || in JavaScript?

In JavaScript, && and || are logical operators used for combining or evaluating expressions. Here are the key differences between them:

Logical AND (&&):

-The && operator returns true if both operands are true.

-It returns false if at least one of the operands is false.

                            let result = (5 > 3) && (2 < 4);
                            console.log(result);  // Output: true

Logical OR (||):

-The || operator returns true if at least one of the operands is true.

-It returns false only if both operands are false.

                            let result = (5 > 3) || (2 > 4);
                            console.log(result);  // Output: true

FOR LOOP

  • finalExpression - This expression is executed after each iteration of the loop. This is usually used to increment a counter, but can be used to decrement a counter instead.

                                if (typeof number !== 'number' || isNaN(number)) {
                                    return "Invalid input. Please provide a valid number.";
                                }

                                // Convert the number to a string to extract its digits
                                const numString = String(Math.abs(number)); // Taking absolute value to handle negative numbers

                                // Check if the number has only one digit
                                if (numString.length === 1) {
                                    return number * 2; // If only one digit, the sum is double the digit
                                }

                                // Calculate the sum of the first and last digits
                                const firstDigit = parseInt(numString[0], 10);
                                const lastDigit = parseInt(numString[numString.length - 1], 10);
                                const sum = firstDigit + lastDigit;

                                // If the original number was negative, make the sum negative as well
                                return number < 0 ? -sum : sum;
                            }
