                        // Function to calculate days left until next Christmas
                        function daysUntilChristmas() {
                            // Get the current date
                            const currentDate = new Date();

                            // Get the current year
                            const currentYear = currentDate.getFullYear();

                            // Set the date for Christmas this year
                            const christmasDate = new Date(currentYear, 11, 25); // Months are 0-based, so 11 represents December

                            // Check if Christmas has already passed this year, if so, set the date for Christmas next year
                            if (currentDate > christmasDate) {
                                christmasDate.setFullYear(currentYear + 1);
                            }

                            // Calculate the difference in days
                            const daysLeft = Math.ceil((christmasDate - currentDate) / (1000 * 60 * 60 * 24));

                            return daysLeft;
                        }

                        // Example usage
                        const daysLeftUntilChristmas = daysUntilChristmas();
                        document.write(`There are ${daysLeftUntilChristmas} days left until next Christmas.`);
Q.10 What is Condition Statement?

The if/else statement is a part of JavaScript's "Conditional" Statements, which are used to perform different actions based on different conditions.

In JavaScript we have the following conditional statements:

  • Use if to specify a block of code to be executed, if a specified condition is true

  • Use else to specify a block of code to be executed, if the same condition is false

  • Use else if to specify a new condition to test, if the first condition is false

  • Use switch to select one of many blocks of code to be executed

  • 5 > 3 evaluates to true.

true && true evaluates to true.

Q.17 What is the result of the expression true && false || false && true?

The expression true && false || false && true involves the logical AND (&&) and logical OR (||) operators.

In JavaScript, the logical AND operator (&&) has higher precedence than the logical OR operator (||). Therefore, the expression is evaluated from left to right, and the result is determined accordingly.

Let's break down the expression:

  1. true && false evaluates to false (because both conditions must be true for the AND operator to result in true).
  2. false || false evaluates to false (because at least one condition must be true for the OR operator to result in true).
  3. Finally, false && true also evaluates to false.
  4. So, the result of the entire expression is false.

Q.20 What is the difference between && and || in JavaScript?

In JavaScript, && and || are logical operators used for combining or evaluating expressions. Here are the key differences between them:

Logical AND (&&):

-The && operator returns true if both operands are true.

-It returns false if at least one of the operands is false.

                            let result = (5 > 3) && (2 < 4);
                            console.log(result);  // Output: true

Logical OR (||):

-The || operator returns true if at least one of the operands is true.

-It returns false only if both operands are false.

                            let result = (5 > 3) || (2 > 4);
                            console.log(result);  // Output: true

FOR LOOP

  • finalExpression - This expression is executed after each iteration of the loop. This is usually used to increment a counter, but can be used to decrement a counter instead.

                                if (typeof number !== 'number' || isNaN(number)) {
                                    return "Invalid input. Please provide a valid number.";
                                }

                                // Convert the number to a string to extract its digits
                                const numString = String(Math.abs(number)); // Taking absolute value to handle negative numbers

                                // Check if the number has only one digit
                                if (numString.length === 1) {
                                    return number * 2; // If only one digit, the sum is double the digit
                                }

                                // Calculate the sum of the first and last digits
                                const firstDigit = parseInt(numString[0], 10);
                                const lastDigit = parseInt(numString[numString.length - 1], 10);
                                const sum = firstDigit + lastDigit;

                                // If the original number was negative, make the sum negative as well
                                return number < 0 ? -sum : sum;
                            }

Q.57 how many type of JS Event? How to use it?

In JavaScript, there are several types of events that can occur in the browser, and you can use them to trigger functions or behaviors in response to user actions or other interactions. Here are some common types of JavaScript events:

  • Mouse Events:
    • click: Occurs when the mouse button is clicked.
    • dblclick: Occurs when the mouse button is double-clicked.
    • mousedown, mouseup: Occur when the mouse button is pressed down or released.
    • mousemove: Occurs when the mouse pointer is moved.
  • Keyboard Events:
    • keydown, keyup: Occur when a key on the keyboard is pressed down or released.
    • keypress: Occurs when a key that produces a character value is pressed down.
  • Form Events:
    • submit: Occurs when a form is submitted.
    • reset: Occurs when a form is reset.
    • change: Occurs when the value of an input element changes.
    • focus, blur: Occur when an element gains or loses focus.
  • Window Events:
    • load: Occurs when the document and its resources (like images and stylesheets) have finished loading.
    • unload: Occurs when the user leaves the page.
    • resize: Occurs when the browser window is resized.
    • scroll: Occurs when the document view or an element is scrolled.
  • Event Delegation:
    • Events like mouseenter, mouseleave, and others related to event delegation can be used to handle events on multiple elements with a single event listener on a common ancestor.
Q.58 What is Bom vs Dom in JS?

BOM (Browser Object Model) and DOM (Document Object Model) are two different but closely related concepts in JavaScript that deal with interacting with web browsers.

  1. DOM (Document Object Model):
    • The DOM is a programming interface provided by browsers that represents the structure of a document as a tree of objects. These objects can be manipulated using JavaScript to change the content, structure, and style of a web page.
    • The DOM represents the HTML or XML document as a tree structure, where each element, attribute, and piece of text in the document is a node in the tree.
    • JavaScript interacts with the DOM to dynamically update or manipulate the content and structure of a web page. For example, you can use DOM methods to create, modify, or delete elements on a page.
  2. BOM (Browser Object Model):
    • The BOM, on the other hand, represents additional objects provided by the browser to interact with the browser itself, rather than the document.
    • The BOM includes objects such as window, navigator, screen, location, history, and document (which is also part of the DOM).
    • The window object, in particular, is a top-level object in the BOM and represents the browser window. It provides methods and properties for manipulating the browser, such as opening new windows, setting timeouts, and handling events related to the window.
    • Unlike the DOM, the BOM is not standardized, and its features can vary between different browsers.
